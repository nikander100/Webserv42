# [Coding Guidelines](./docs/guidelines.md)

## Naming Conventions

- Use camel case for functions and variables.
	- Example: `myFunction`, `myVariable`

- Use Pascal case for classes.
	- Example: `MyClass`

- Use snake case for function parameters.
	- Example: `my_parameter`
